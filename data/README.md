## CSV из «Расходы ОК» — формат

Tab-separated, UTF-8, заголовок:
```
"Дата"  "Сумма"  "Валюта"  "Счет"  "Статья"  "Группа статей"  "Комментарий"
```

- `Дата` — `YYYY.MM.DD`
- `Сумма` — отрицательная для расходов (`-120`, `-3000`)
- `Валюта` — **пустая, потому что «Расходы ОК» работает в одной валюте**; считаем что всё в RSD
- `Счет` — почти всегда `"Мои деньги"`
- `Статья` — категория (Еда, Спорт, Транспорт, Жилье, Продукты, Здоровье, Коммуналка, Техника, Досуг, Одежда, Красота, Дом)
- `Группа статей` — почти везде пустая
- `Комментарий` — свободный текст

## Зачем эта папка отдельно

- **Privacy**: личные финансовые данные не должны попадать в git history.
- **Размер**: бинарные .PNG и .xlsx не нужны в commits.
- **Гибкость**: можно подложить любые источники (банковские выписки, ещё CSV) — они автоматически gitignored.

## Бэкапы

`data/legacy/backups/` сохраняется (но содержимое gitignored). Туда пишут скрипты `tools/excel/backup.py` и `restore.py`.

Долговременный бэкап — отдельным механизмом (например, копия в iCloud Drive).
